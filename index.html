Storage File Assistant

 

This is a simple and easy to use file manager for Android users to provide more convenience.

Support common file operations, can open, browse, copy, move, delete, paste, new, rename, compress, decompress files.

Support a variety of images, video, audio and other media file formats, and commonly used other file formats.

You can view the details of a file or group of files.

It can also help you when you need to find a file.


In addition to basic file handling capabilities, it also includes:

Supports external memory: browse and operate internal and external memory, including SD card and U disk

Bookmarks: Quick access to frequently used files or folders

File remarks: add explanatory text to important files

Recycle Bin: Recovering precious files that have been deleted

Application analysis: See which applications take up more storage space
